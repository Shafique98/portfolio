Project 01: Inventory & Stock Management System

Problem

Inefficient inventory tracking with manual processes leading to stock discrepancies and delayed updates across systems

Solution

Built Angular PWA interfaces for item listing and stock control. Developed Spring Boot microservices integrated with SAP for real-time stock updates. Connected PWA with Intershop back office.

Tech Stack

Angular PWA Intershop Spring Boot REST APIs SAP Integration Microservices

Contribution
